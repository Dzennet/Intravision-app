Intravision App

Это SPA - приложение написано для отображения, создания и изменения заявок. Каждая заявка предусматривает за собой добавление комментария, названия, описания, а также смену статуса заявки и ее исполнителя.

На проекте были использованы следующие технологии: React, Redux, также есть взаимодействие с API
